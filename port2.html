Overview.

In addition to the logo on my website, I also fully created this responsive website from scratch using HTML and CSS. I personally feel very comfortable with the HTML, CSS, and even javaScript language.

In the example above, you can see one of the first websites I ever built in school using simple HTML and CSS. The website itself is supposed to be an interactive educational type website, that allows any visiting users to learn more about HTML and CSS in the form of a game. With this in mind, the index page offers the user the ability to login or create a new account that will keep track of the user’s progress and achievements. The index page also contains a navigation bar that allows the user to click through other pages on the site like the gallery, about us, and contact us page. Not only this, but once the user logs in successfully or registers on to the website he is directed to the game portion of the site. Using a few customize buttons and a little javaScript code I allow the user to click through the website and learn more about HTML and CSS by changing the text, color, and font on the screen depending on the different buttons they click on. The webpage also educates the user on other HTML and CSS concepts like, html tags, selectors, and classes. The about Us page, allows the user to learn more about the creator and his goals, as well as a donation section that allows the user to donate and support the website. Lastly, the contact us page offers the user the ability to contact the website creator and submit any questions or feedback he has regarding the website.

Skills Learned

Some of the skills learned from this project were.

This particular HTML project taught me how to create a responsive website that can drastically change what is displayed on the screen, depending on what the user clicks on to trigger these changes. Not only this, but I also learned how to effectively use the <div> tags to properly position data on the screen and avoid clutter and misdirection for the user. I also learn about some visual queues that can be used to guide the user through the website like zoom in object, color change when the mouse hovers, and also multiple buttons for the user to click on and explore for himself.
